
# PDF Merger & Splitter

## Project Overview

PDF Merger & Splitter is a Python-based utility application that allows users to merge multiple PDF files into a single PDF document and split a PDF document into individual pages. The project is developed using the PyPDF2 library and demonstrates practical file handling concepts in Python.

## Features

* Merge multiple PDF files into one PDF
* Split a PDF into separate pages
* Display PDF information
* File existence validation
* Exception handling for invalid inputs
* Menu-driven console application

## Technologies Used

* Python 3
* PyPDF2 Library
* File Handling
* OS Module
* Exception Handling

## How to Run

### Install Required Library

```bash
pip install PyPDF2
```

### Run the Application

```bash
python pdf_tool.py
```
